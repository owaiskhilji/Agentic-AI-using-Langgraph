# Agentic AI using LangGraph

This repository demonstrates how to build **Agentic AI workflows using LangGraph and Large Language Models (LLMs)**.

The project explores different workflow patterns used in modern AI systems, including structured prompts, prompt chaining, and LLM-driven task execution.

## 🚀 Features

* Implementation of **LLM Workflows**
* **Prompt Chaining** examples
* Structured **Agentic AI pipelines**
* Jupyter Notebook demonstrations for learning and experimentation
* Python implementation for prompt chaining logic

## 📂 Project Structure

* `1.bmi_workflow.ipynb` – Example workflow for BMI calculation using LLM logic
* `2llm_workflow.ipynb` – Demonstration of LLM-based workflow execution
* `3prompt_chiaini.ipynb` – Prompt chaining examples for sequential reasoning
* `promptchain.py` – Python implementation of prompt chaining logic
* `upsc_essay_workflow.py` - Essay Checking 

## 🧠 Technologies Used

* Python
* LangGraph
* Large Language Models (LLMs)
* Jupyter Notebook

## 🎯 Purpose of this Project

The goal of this repository is to **understand how Agentic AI systems work** by experimenting with different workflow patterns and prompt engineering techniques.
